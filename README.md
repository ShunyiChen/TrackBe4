# TrackBe4
A web system for vehicle administration office.

## The following is the technique used for this project:
Java1.8

Vaddin8.2.0

PostgresSQL9.4/H2

Jetty/Tomcat

SpringBoot

ImageJ

Caffeine cache

Jbcrypt encryption

Truelicense

ireport

Maven3.5

Log4j

## Run and package
spring-boot:run

mvn clean package
